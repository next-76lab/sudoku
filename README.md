# ✨ Starlight Nanpure ✨

可愛いナンプレゲーム！

## 特徴

- 🌙 **動く背景**: 顔付きの雲・笑顔の月・きらめく星・流れ星
- 🎮 **100ステージ**: 難易度が徐々に上昇
- 📱 **スマホ完全対応**: タッチ操作最適化
- 💾 **進捗保存**: LocalStorageでクリア状況を記憶
- 🎯 **ステージセレクト**: 好きなステージを選択可能

## 実行方法

```bash
pip install streamlit
streamlit run app.py
```

ブラウザで `http://localhost:8501` にアクセス！

スマホからは Network URL を使用してください。

## 操作方法

1. **マスをタップ** → 選択
2. **数字パレット** → 数字を入力
3. **全マス正解** → ステージクリア！

## ボタン

| ボタン | 説明 |
|--------|------|
| 💡 ヒント | 空きマスに正解を1つ入れる |
| 📋 選択 | ステージセレクト画面を開く |
| 🔄 リセット | 現在のステージをやり直す |
